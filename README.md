# python-notebook-manager--Arwa-Meari-
Personal Notebook Manager

A simple command-line application to manage personal notes with titles, content, tags, and automatic timestamps. Notes are saved locally in a JSON file.

Features:

  [1]Add new notes with title, content, and optional tags. <br>
  [2]List all saved notes.<br>
  [3]Search notes by keyword in title or content.<br>
  [4]Filter notes by tags.<br>
  [5]Edit existing notes.<br>
  [6]Delete notes.<br>
  [7]Automatic saving and loading from notes.json.

Requirements:
  Python

How to Run:

  Make sure Python is installed on your system.<br>
  Save the project files in a folder.<br>
  Open a terminal or command prompt in the folder.<br>
  Run the application<br>
  Follow the on-screen menu to manage your notes.<br>
  When you choose to exit, all notes will be automatically saved.<br>
