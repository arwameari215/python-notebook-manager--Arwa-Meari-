# python-notebook-manager--Arwa-Meari-
Personal Notebook Manager

A simple command-line application to manage personal notes with titles, content, tags, and automatic timestamps. Notes are saved locally in a JSON file.
Features:
  [1]Add new notes with title, content, and optional tags.
  [2]List all saved notes.
  [3]Search notes by keyword in title or content.
  [4]Filter notes by tags.
  [5]Edit existing notes.
  [6]Delete notes.
  [7]Automatic saving and loading from notes.json.

Requirements:
  Python

How to Run:
  Make sure Python is installed on your system.
  Save the project files in a folder.
  Open a terminal or command prompt in the folder.
  Run the application
  Follow the on-screen menu to manage your notes.
  When you choose to exit, all notes will be automatically saved.
